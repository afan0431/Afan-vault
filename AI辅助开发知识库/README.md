# AI 辅助开发知识库

AI 编程助手的能力上限，很大程度上取决于它能获取多少**准确的项目上下文**。把架构决策、业务规则、领域术语这些"隐性知识"结构化存储，让 AI 在需要时按需检索，是目前最实用的提升 AI 辅助开发质量的方式。

这套文档记录了从零搭建一套面向开发场景的 RAG（检索增强生成）知识库的完整实践，包括内容设计、技术选型、工程实现到长期维护。

## 快速上手

```bash
# 安装 Ollama 并拉取 Embedding 模型
ollama pull bge-m3

# 启动 Qdrant
docker compose up -d

# 建库（首次全量）
python index_knowledge.py

# 查询
python query_knowledge.py "你的问题"
```
